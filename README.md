# Desafio-POO
Desafio: Aprenda praticando a Programação Orientada a Objetos

Sejam todos bem recebidos no desafio: EXPLORANDO NA PRÁTICA O CONCEITO DE PROGRAMAÇÃO BASEADA EM OBJETOS. Esse desafio é fruto de uma colaboração com a plataforma de ensino online DIGITAL INNOVATION ONE e está disponível gratuitamente para a comunidade de programadores Java. O principal objetivo é colocar em prática uma das principais técnicas da POO: ABSTRAÇÃO, ENCAPSULAMENTO, HERANÇA E POLIMORFISMO, por meio de um projeto em Java.
